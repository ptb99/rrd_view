Django server to view RRD charts from a voltage logger

This is intended as a complete package to turn a Raspberry Pi into a simple data logger (using an ADC, such as on the Pimoroni Automation pHat).

Currently, this just handles the web server to display charts from RRD that have been recorded by the other components. The Django DB is used to hold the list of RRD duration specs that define the charts we want to offer/display. Much of the charting is hard-coded for now.

See https://github.com/ptb99/rrd_view for source code.
